# UbuntuX - Social Savings for the Digital Age

UbuntuX is a cross-border Adashi (Social Savings) mobile application built for the **Enyata Hackathon**. It leverages technology to modernize traditional African savings circles ("Adashi" or "Ajo") for a global, trust-less digital economy.

## Value Proposition

### 1. Social Savings (Adashi)
Digitalize communal saving circles with automated payouts, group management, and transparent tracking. UbuntuX makes it easy for friends, family, and communities to save together, regardless of where they are.

### 2. Cross-Border Payments
Seamlessly send and receive funds across borders. UbuntuX integrates robust payment gateways to facilitate low-cost, high-speed international transfers within the app's ecosystem.

### 3. AI-Driven Trust Engine
The **UbuntuX Trust Engine** uses Python-powered AI models to generate dynamic trust scores based on saving habits, consistency, and social verification. This enables secure participation in groups without traditional credit history.

## Tech Stack
- **Frontend**: Flutter (Clean Architecture)
- **Backend**: Python FastAPI (AI Engine)
- **Database**: PostgreSQL / Firestore
- **AI**: Scikit-learn / TensorFlow for Trust Scoring
